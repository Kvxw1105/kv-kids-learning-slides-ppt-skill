# 儿童学习与演讲课件工坊

This Skill creates child-friendly PPTX presentations through a complete workflow:

1. Identify classroom or child speech scenarios.
2. Define age, difficulty, vocabulary, bilingual mode, and output requirements.
3. Build content structure, visual direction, and character continuity.
4. Generate editable PowerPoint layouts using native components and visual assets.
5. Create Markdown and Word speaker scripts when required.
6. Run geometry checks and rendered visual review before delivery.

Core principles:

- Keep screen content separate from teacher/speaker notes.
- Prefer editable PowerPoint objects for structure.
- Use AI illustrations only where they add narrative value.
- Use code-generated SVG/components for reusable decorations.
- Validate typography, spacing, overlap, visual density, and consistency.

For commercial-quality output, use the full references, scripts, and assets included in this repository.
