# Commercial aesthetic review

Run this after rendered visual review. The reviewer must answer per deck:

- What is the first visual memory after 3 seconds?
- Is the deck distinguishable from generic AI templates?
- Which slide is the emotional peak?
- Do English and Chinese feel designed together?
- Are there more than three meaningful layout rhythms?
- Does every large blank area have a role?
- Which slide would a paying user complain about first?

Score 0-10:

- first impression
- visual hierarchy
- character/emotion
- page rhythm
- typography integration
- style uniqueness
- commercial finish

Commercial-premium acceptance: overall >= 8, no major dimension < 7.
