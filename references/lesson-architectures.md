# 课程架构库

## 概念建构型

适用数学、科学、语法。具体情境 → 操作/观察 → 多个案例 → 抽象规律 → 反例 → 练习 → 迁移。

## 故事探索型

适用语文、阅读、历史启蒙、品德。角色目标 → 阻碍/疑问 → 线索 → 关键发现 → 价值或知识结论 → 表达与迁移。故事不能吞没知识目标。

## 技能训练型

适用计算、识字、拼音、单词。诊断 → 微讲解 → 引导练习 → 高频短反馈 → 混合练习 → 错题纠正 → 达标检测。

## 实验观察型

适用科学与自然。提出问题 → 预测 → 材料与步骤 → 观察记录 → 解释 → 控制变量/误差 → 生活联系。

## 项目任务型

适用综合实践。真实任务 → 评价标准 → 分工与资源 → 阶段成果 → 展示 → 反思。PPT 同时承担任务说明和过程支架。
