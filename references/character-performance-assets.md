# Character performance assets

For character-led children speeches and story lessons, build a character performance plan before slides. Minimum poses:

1. greeting / waving
2. showing object
3. observing closely
4. action pose
5. waiting or thinking
6. surprise / discovery
7. happy celebration
8. closing / thanks

If image generation is unavailable, use code components and simple silhouettes, but state the downgrade. Do not repeatedly reuse the same neutral standing pose as a substitute for different story states.

Every character asset should preserve age, hair, outfit, signature accessory, face style, and palette.
