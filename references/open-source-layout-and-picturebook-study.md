# 开源项目学习记录与许可边界

## 可吸收的工程经验

### PptxGenJS

- 使用母版、形状、SVG、亚洲字体、动画媒体和大量演示样例构建可编辑 PPTX。
- 学习其“对象化生成”和演示用例，不把整页栅格化。
- 许可：MIT。

### OpenAI slides helpers

- 使用文本测量、自动字号、图片 contain/crop、对齐和重叠检测。
- 本 Skill 可调用宿主环境中的 helper，但不把宿主私有代码复制进分发包。

### Yoga

- 学习 Flexbox 的内容测量、主轴/交叉轴、收缩与分配思想。
- 本 Skill 只独立实现固定舞台的语义区域规划，不捆绑 Yoga。
- 许可：MIT。

### opentype.js

- 学习使用真实字体度量、字距、字形边界和 advance width 进行精确排版。
- 在 Node 环境存在字体文件时可作为可选增强；缺失时继续使用当前测量降级。
- 许可：MIT。

### Rough.js 与 SVG.js

- Rough.js 提供手绘线条、填充和 SVG 路径；SVG.js 提供轻量 SVG 操作与动画。
- 本 Skill 独立实现轻微不规则路径和 SVG 组件，不复制库代码；宿主可选安装后替换为更丰富效果。
- 许可：MIT。

### Book Dash

- 公开 200+ 儿童绘本的电子版、印刷版和 working source files，可用于研究绘本构图、字体、图文关系和翻译版式。
- 不默认把 Book Dash 插画或源文件打进 Skill；若实际复用必须署名原作者与 Book Dash。
- 内容许可：CC BY 4.0。

### Scribus

- 学习桌面出版中的文本框、边距、出血、字体、双向语言与印刷布局思路。
- 不捆绑 Scribus 代码。
- 许可按上游项目声明处理。

### OpenMoji

- 可用于研究一致的图标风格、中央数据表、导出脚本和自动一致性测试。
- 默认 Skill 不直接打包 OpenMoji 图形，避免 CC BY-SA 传播要求影响默认资产许可。
- 图形：CC BY-SA 4.0；代码：LGPL-3.0。

## 默认分发策略

- 新增脚本均为独立实现。
- 默认组件只使用 Skill 自己生成的 SVG。
- MIT 项目可在未来有选择地引入，但必须保留许可证与版权声明。
- CC BY / CC BY-SA 素材只有在明确记录署名和衍生许可时才允许进入交付物。
- AGPL、专有和仅限个人使用的代码、模板不得混入默认包。
