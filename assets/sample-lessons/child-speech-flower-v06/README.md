# Child Speech Flower v0.6

This sample demonstrates a mixed visual pipeline:

- a small number of approved AI/source hero illustrations;
- native PowerPoint flower, weather, watering and before/after scenes;
- code-generated paper-theater stickers and page accents;
- a conservative `motion_plan.json`;
- visual-density metadata in each slide.

Use `scripts/compose_visual_components.py` to regenerate or replace the component plan.
